# Profile Card – Stage 1B

## What this project is
This is a simple profile card built as part of the HNG Stage 1B task.  
It shows basic user information like name, bio, avatar, social links, hobbies, dislikes, and a live time in milliseconds.

The goal was to focus on clean structure, accessibility, responsiveness, and making sure everything works with the required test attributes.

## How to run this project locally
If you want to run it on your machine:

1. Clone this repo:
   git clone https://github.com/Sherifat-Aduku/hng-stage1b-profilecard.git

2. Go into the folder:
   cd profile-card

3. Open `index.html` in your browser.

That’s it — no setup needed.

## What I used
Just the basics:
- HTML
- CSS
- JavaScript

No frameworks, no libraries.

## Features
- Shows user name, bio, and profile picture
- Displays social links (Twitter, GitHub, LinkedIn, Website)
- Lists hobbies and dislikes
- Shows live epoch time in milliseconds
- Fully responsive on mobile, tablet, and desktop
- Keyboard-friendly and accessible

## About the time display
The time is generated using `Date.now()` and updates on the page to show the current time in milliseconds.
